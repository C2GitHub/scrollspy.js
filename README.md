# `scrollspy.js`滚动监听

##`scrollspy.js`滚动监听插件使用介绍

### 插件引用
```javascript
<script src="./js/scrollspy.js"></script>
```


### html页面框架布局

> * `class="scrollspy"`为页面滚动监听项目
> * `id="scrollspy"`为页面监听导航条

```javascript

<ul>
<li class="scrollspy">main section1</li>
<li class="scrollspy">main section2</li>
<li class="scrollspy">main section3</li>
<li class="scrollspy">main section4</li>
<li class="scrollspy">main section5</li>
</ul>
<ol>
<li>index1</li>
<li>index2</li>
<li>index3</li>
<li>index4</li>
<li>index5</li>
</ol>
```
